# Red Hat Advanced Cluster Management

### This repository provides sample patterns that can be used with Red Hat Advanced Cluster Management

## Cluster LifeCycle Sample Patterns
### User pattern 1
This pattern depicts how to provide self serve Hosted Control Plane clusters via the Red Hat Advanced Cluster Management console.
  * Multitenant user solution

### User pattern 2
This pattern depicts self served Hosted Control Plane clusters using templates, this approach does not expose the Cloud credentials to the end user.
  * Exposing kubeadmin will be optional
  * Can be used with quotas
  * Multitenant user solution
